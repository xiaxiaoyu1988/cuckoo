===================
Cuckoo Sandbox 中文文档
===================

Cuckoo Sandbox 是一个开源的自动分析可疑文件的软件。它通过在独立环境内使用自定义组件来监控恶意进程的行为。

本文档旨在说明如何配置、使用和定制Cuckoo
